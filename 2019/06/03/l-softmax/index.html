总结

看了三个文章,都是通过减小内类间距来达到效果.减小内类间距的途径都是构建$\psi(\theta)$代替$cos(\theta)$,当然要保证$\psi(\theta) < \cos\theta$.

之前的L softmax和A softmax为了保证$\psi(\theta)=cos(m\theta) < \cos\theta$还使用了分段函数,比较麻烦.然后AM softmax就比较简单粗暴了.
