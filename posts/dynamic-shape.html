    with R.dataflow():
        n = T.int64()
        c: R.Tensor((n, 10)) = a + b
        R.output(c)
    return c

而根据relax shape设计文档下面这种情况应该是无法支持的:

@R.function
def fn2(a: R.Tensor(("n", 10), 'float32'), b: R.Tensor((1,), 'float32')):
    with R.dataflow():
        n = T.int64()
        c = a + b
        cshape: R.Shape() = R.shape_of(c)
        d = R.reshape(c, [1, cshape[0], cshape[1], 1])
        R.output(d)
    return d

我在思考是不是应该有一种直接基于数据流的方式来添加symbolic shape的信息?
